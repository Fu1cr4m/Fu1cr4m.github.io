文件管理

cat

用于连接文件并打印到标准输出设备上

chattr

用于改变文件属性

chgrp

用于变更文件或目录的所属群组

chmod

更改文件调用权限

chown

root账号指定文件的拥有者改为指定的用户或组

cksum

检查文件的CRC是否正确。确保文件从一个系统传输到另一个系统的过程中不被损坏。

cmp

用于比较两个文件是否有差异

diff

逐行方式比较文本异同处。

diffstat

显示统计diff的结果

file

辨识文件类型

find

在指定目录下查找文件

git

文字模式下的文件管理员

gitview

观看文件的内容，它会同时显示十六进制和ASCII格式的字码

indent

调整C原始代码文件的格式

cut

显示每行从开头算起 num1 到 num2 的文字

in

为某一个文件在另外一个位置建立一个同步的链接
